# Anime-HandFX

A simple hand-tracking web project that lets you trigger **Naruto** and **Sasuke** inspired powers using your hands through your webcam.

## How to Use

1. Open `index.html` in your browser.
2. Allow access to your webcam.
3. Show your hands to the camera.
4. Open either hand to activate powers.

## Controls

### Right Hand Open

* Activates **Naruto power**

### Left Hand Open

* Activates **Sasuke power**

### Blue Lines

* Displays **hand tracking markers** so you can see that the detection is working

## Notes

* Glow effects were removed for better compatibility across devices.
* Works best in a well-lit environment.
* Keep your hands visible to the camera for accurate tracking.

## Tech Stack

* HTML
* CSS
* JavaScript
* Webcam Hand Tracking
